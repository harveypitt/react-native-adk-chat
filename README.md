# React Native ADK Chat

Clean, minimal chat UI with monochrome design and rounded elements.

## Features

- ✅ Zen-like minimal design
- ✅ Monochrome color palette
- ✅ Rounded message bubbles
- ✅ Airplane send icon
- ✅ Canned responses for prototyping
- ✅ TypeScript support
- ✅ Expo ready

## Quick Start

1. Install dependencies:
   \`\`\`bash
   npm install
   # or
   pnpm install
   \`\`\`

2. Start the development server:
   \`\`\`bash
   npx expo start
   \`\`\`

3. Open in Expo Go app or browser

## Design Philosophy

- **Minimal**: Only essential elements, no decoration
- **Monochrome**: Clean grayscale palette
- **Rounded**: Soft, friendly 20px border radius
- **Typography-focused**: Clear hierarchy through text
- **White-label ready**: Easy to customize

## Components

### MessageBubble
- Rounded message containers
- User messages: Light gray background
- AI messages: White with subtle border
- Clean timestamps and role labels

### ChatInput
- Rounded text input (24px radius)
- Matching send button height
- Simple airplane icon
- Monochrome styling
